### Bitfinex Market Prices Blocklet for i3

![screenshot](/readme_assets/screenshot.png)

Sample configuration:

```
[bitfinex]
command=/path/to/index.js
symbol=BTCUSD
interval=30

[bitfinex]
command=/path/to/index.js
symbol=ETHUSD
interval=30

[bitfinex]
command=/path/to/index.js
symbol=LEOUSD
interval=30
```
